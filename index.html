School Administration Management

This system is built with laravel framework and MySQL database using templates from the Admin LTE.

The features of this system globally include:

  • Multi user (Admin, Officer, Student)
  • Managing Master Data (School Identity, Teachers, Students, Study Groups, etc.).
  • Manage Mail Management (Incoming Mail and Outgoing Mail).
  • Manage Student Payments
  • Managing Student Savings

  • Multi user (Admin, Officer)

  • Managing Letter Classification
  • Print incoming and outgoing mail agendas
